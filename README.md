QA Automation CI/CD Demo

Project Overview

This project demonstrates a simple QA automation workflow integrated with a CI/CD pipeline using GitHub Actions and Docker.

The goal of this project is to show how automated tests can be executed every time code is pushed to the repository, ensuring that the application remains stable and functional.

The project contains:

A simple Python application

Automated tests using pytest

A Docker container for running the application

A GitHub Actions pipeline that automatically runs tests
